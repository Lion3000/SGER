Purpose of this application:

SGER stand for Results Management System. This application replaces the pen and paper sheets used on a boat by the Race Committee, which is not practical.

With this application you can:

  • Organize a Pairing-List
  • Organize an Event from its creation to the management of the results
  • See the results of an event
  • Export the results to PDF and HTML
  • Send the exported files via Bluetooth

    • Guide: Add a Crew

    To add a Crew:

    1. Open the side-bar menu
    2. Press the Crew-list button
    3. Push the (+) button
    4. Add the Crew's informations then save

    Guide: Organizing an Event

    To organize an Event:

    1. You need to already have added crews on the Crew-list
    2. Open the side-bar menu
    3. Press the Event-list button
    4. Push the big (+) button
    5. Add the Event's informations and the crews participating
    6. Create the Event

    Guide: Organizing a Phase

    To organize a Phase:

    1. You need to already have created an Event
    2. Open the side-bar menu
    3. Press the Event-list button
    4. Select the Event where you want to organize the Phase
    5. Push the small (+) button
    6. Enter the name of the Phase
    7. Choose whether the Phase is a RoundRobin or not by checking the RoundRobin button
    8. Select the Phase where you want to enter the Pairing list
    9. Press the (+) button to add a Flight
    10. To add a Match press on the Flight's Match
    11. Select the 2 Crew racing and attribute boat to each racer
    12. Choose the winner of the Match buy pressing on the crew's name then save
    13. If you want to add a Penalty to a Crew press on their name, choose the penalty and add its reason then save

    Guide: Export and Transfer Results

    To export Results:

    1. You need to already have created an Event
    2. Open the side-bar menu
    3. Press the Event-list button
    4. Select the Event then/or the Phase you want to export
    5. Press the HTML or PDF button to create a PDF or a HTML file
